# $IGLOO — The Future is Frozen

$IGLOO is a high-performance, immersive Web3 landing page concept. It features interactive 3D elements, real-time data visualizations, and a suite of DeFi utility mockups.

## 🚀 Key Features

- **Interactive 3D Background**: Built with **Three.js** to create a fixed, responsive 3D environment.
- **Scroll-Triggered Animations**: Powered by **GSAP** and **ScrollTrigger** for smooth, cinematic transitions.
- **DeFi Utilities**: 
  - **Staking Calculator**: Interactive tool with custom sliders to estimate rewards.
  - **Live Price Chart**: A dynamic canvas-based chart visualization.
  - **Tokenomics Dashboard**: Responsive donut chart and detailed distribution breakdown.
- **Web3 Integration UI**: Built-in wallet connection modal and "Connected" state management.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Properties), JavaScript (ES6+).
- **3D Engine**: [Three.js](https://threejs.org/).
- **Animation**: [GSAP](https://greensock.com/gsap/).
- **Typography**: Bebas Neue & JetBrains Mono via Google Fonts.

## 📦 How to Use
1. Clone the repository: `git clone https://github.com/your-username/igloo-token.git`
2. Open `index.html` in any modern web browser.
3. To view the 3D effects, ensure you are running a local server (e.g., Live Server extension in VS Code).
